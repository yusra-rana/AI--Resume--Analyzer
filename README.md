# AI Resume Analyzer 🧠📄

A smart Python desktop application that analyzes resumes and gives ATS score, skills match, missing skills, and grammar suggestions.

## 🚀 Features

✅ Upload Resume PDF  
✅ Extract Resume Text  
✅ ATS Score Checker  
✅ Detect Technical Skills  
✅ Show Missing Skills  
✅ Grammar Suggestions  
✅ Simple GUI using Tkinter  

## 🛠 Tech Stack

- Python
- Tkinter
- PyPDF2
- TextBlob
- NLTK
